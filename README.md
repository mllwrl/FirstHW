# Fibonacci Sequence Project

A simple console application developed as part of a programming task to demonstrate Git workflow and algorithm optimization.

## Stages:
1. **Hello World**: Initial project setup.
2. **Recursive approach**: Calculating $n$-th number (sequence starts with 1, 1).
3. **Sequence adjustment**: Changing the base case to (0, 1).
4. **Iterative optimization**: Printing the full sequence up to $n$ for better performance.
